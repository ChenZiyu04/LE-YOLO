# LE-YOLO
https://doi.org/10.5281/zenodo.19313659

## 1. Description
This repository contains the code and data associated with the paper "LE-YOLO:Task-driven curve enhancement with attention-guided hostnet for low-light object detection.". 

## 2. Dataset Information
- **Source:** Downloaded from Kaggle
- **Format:** .jpg

## 3. Code Information
- **Language:** Python
- **Main Scripts:**
    - `custom_module.py`: Cleans and prepares the raw data.
    - `new_custom_module.py`: Prepares the raw data.
    - `zerodce_module.py`: Prepares the zerodce.
    - `leyolo_cbam.yaml`: The main script to train the model.
    - `train_val_split.py`: Calculates metrics and generates plots.
